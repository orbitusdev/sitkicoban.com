## 1.0.0

### Major Changes

- 9e404d8: feat: Initial Project Release (v1.0.0)

  This changeset marks the inaugural public release of all packages in the Orbitus monorepo.

  ### What Changed

  This is the first version of the packages. It includes the entire foundational codebase, project structure, build tooling, and initial feature sets for all applications and libraries.

  ### Why This Change Was Made

  This release establishes the first stable baseline (v1.0.0) for all packages, allowing for public consumption, integration, and further development. It formalizes the project's structure and makes it available for users and contributors.

  ### How to Update

  As this is the first release, there is no upgrade path. Consumers can start by installing the packages. For example:

  `pnpm add @orbitusdev/core@latest`
